C'est quoi?

Design et intégration web d’un site événementiel fictif, fait dans le cadre des cours de design d’expérience et conception web lors de ma quatrième session du programme

Type de projet

Ce projet est une projet scolaire.

Site responsive

Dans ce projet j’ai utilisé des techniques pour créer un site responsif pour qu’il s’adapte aux différentes tailles d’écrans de nos jours.

Documentation

Tout au long de l’intégration de ce site, j’ai fait de la documentation, tant pour les utilisateurs que pour les développeurs.
